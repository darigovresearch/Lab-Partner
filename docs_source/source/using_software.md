# Using the software

## Opening the page

If you have followed the instructions in [Initial installation](installation.md) now you can navigate to the local URL which can be found on <a href="http://0.0.0.0:8765/" target="_blank">http://0.0.0.0:8765/</a> or <a href="http://0.0.0.0:8765/" target="_blank">http://localhost:8765/</a>

## Changing settings

You can then set the settings for how often you wish for readings to be set or if you want to temporarily stop taking readings.

## Previewing Data

You can see a table of photos, when they were taken and what the reading was.
